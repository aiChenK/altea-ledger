# CHANGELOG / 变更日志

## [v1.3.2] - 2026-06-22

### Fixed / 修复
- **修复手机端白屏及静态资源 404 故障**：
  - 优化 Service Worker 缓存策略：将 `/` 和 `/index.html` 调整为 **Network-First (网络优先)** 策略。在网络可用时，优先获取最新版 HTML，避免由于前端构建更新（Hash 静态资源改变）而导致客户端缓存旧版 HTML 请求不存在的 JS 资源导致 404 白屏；离线时降级使用本地缓存，保持 PWA 特性。
  - 更新 Service Worker 缓存名称（`CACHE_NAME`）至 `v1.3.2`，自动强制清理和重载旧版失效缓存，根治因静态资源文件名变更引发的加载中断。

## [v1.3.1] - 2026-06-21

### Added / 新增
- **主表格首列金币总和展示**：

  - 采用金色微透明徽章样式，完美融入现有的玻璃拟态暗色主题，直观且不占额外空间，方便快速掌握全账号金币存量。

## [v1.3.0] - 2026-06-21

### Added / 新增
- **每日重置金币快照与历史分析系统**：
  - 核心重置逻辑升级：在每日重置（自动触发或手动强制重置）发生时，自动为各角色生成资产金币快照，并智能计算与前一日快照的增减额度（`change`）。
  - 精确账期日期划分：利用早上 9 点重置点向前偏移 12 小时的计算规则，使得重置时刻生成的金币记录能够准确归属于昨日的资产账期（例如 6 月 21 日早上重置，生成 6 月 20 日的账目历史）。
  - 角色重命名无缝迁移：配置页面修改角色名称时，金币历史快照（`data.goldHistory`）中的角色名称同步更改。
  - 新增金币账本一键清空功能（`POST /api/gold-history/clear`），包含防误触确认机制。
- **金币账本与统计多维抽屉**：
  - 主界面 Header 新增 🪙 按钮，一键划出磨砂玻璃风格的金币账本侧边抽屉。
  - **每日变化 Tab**：按日期倒序展示每天所有角色金币总计及总盈亏，支持展开查看各角色的金币数量与增减值。
  - **每月汇总 Tab**：按月份（YYYY-MM）聚合分析，展示月度所有角色的总净变化以及单个角色的月度净金币变化。
  - **角色分析 Tab**：一键计算该角色当前金币持仓、历史累计收入总额、历史累计支出总额、净变化，并列出最近 7 次的变化轨迹线。

## [v1.2.2] - 2026-06-18

### Fixed / 修复
- **彻底根治磨砂玻璃内部重绘导致的 CPU 高占用与卡顿问题**：
  - 移除了已完成/进行中进度格子在静置状态下的 `infinite`（无限循环）发光闪烁动画，改用静态的高发光效果，彻底消除了静置页面时的 CPU/GPU 重绘损耗，使静置 CPU 占用归零。
  - 解决了在 Edge 浏览器中将网页“安装为独立 App”运行时，由于无限重绘触发的 GPU 高频磨砂玻璃高斯模糊运算造成的系统其他窗口（如视频播放）卡顿掉帧问题。
  - 保留了交互微动效：仅在鼠标 `:hover` 悬浮到格子上时，提供平滑过渡的发光变亮微动效，实现性能与美感的完美平衡。

## [v1.2.0] - 2026-06-17

### Added / 新增
- **游戏收支记账系统**：
  - 主界面新增高颜值玻璃拟态收支记录面板，支持快速录入每一笔账目流水的增减。
  - 细分“收入”与“支出”方向，并支持普通收支和“买金/出金（卖金）”等游戏特色交易类型。
  - 出金/卖金时支持输入总计金额、金币数、出金兑换比例占位（如 23.6 代表每￥1兑换的金币数量）及金币手续费。
  - 记账时备注支持选填，留空时由系统自动根据交易类型补全默认智能备注。
- **收支流水统计与年/月汇总**：
  - 主界面顶部实时计算并直观展示历史总收入、总支出及结余统计。
  - 记账面板内提供专门的“统计”选项卡，自动按年份、月份进行折叠汇总，多级计算年/月收入、支出与净结余。
- **数据导入导出**：
  - 支持一键将收支历史序列化导出为 `.txt` 逗号分隔多行纯文本备份文件。
  - 导入支持高兼容性解析（兼容中英文逗号、缺省日期 `-` 使用今天、智能捕获格式错误并给出明确弹窗提示）。

## [v1.1.5] - 2026-06-16

### Added / 新增
- **页面展示控制**：在配置抽屉中增加了“展示与自定义项目”区域，允许勾选/取消勾选来灵活控制默认的“金币”、“时装收集”、“黄金鹅”和“角色代办”在页面及历史快照中的显示与隐藏。
- **自定义项目配置**：支持用户单独配置自定义项目，允许随时添加、删除、重命名和拖拽排序这些项目。可选的自定义项类型包括：
  - **数字**：渲染为数字输入框，可自由输入。
  - **时间**：渲染为到期计时卡片，点击可通用弹窗快捷选择或清空到期时间。
  - **进度**：支持填入指定的阶段数（1-10阶段），渲染为点状进度块，点击可自适应循环累加。
- **配置数据自动对齐**：后端数据对齐逻辑适配了新增资产的多种初始化场景，确保自定义项在新角色或既有角色的配置更新时，其数据结构能够智能兼容与平滑补齐。

## [v1.0.0] - 2026-06-15

### Added / 新增
- **多用户验证与物理隔离**：
  - 新增多用户支持。通过在系统运行前配置 `USERS_AUTH` 环境变量，支持不同用户以独立昵称和独立密码登录。

  - 兼容单人密码模式（由 `ADMIN_PASSWORD` 控制，昵称显示为“管理员”）。

  - 浏览器通过 `LocalStorage` 记住登录凭证，免除重复输入。
