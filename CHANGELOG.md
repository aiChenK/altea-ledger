# CHANGELOG / 变更日志

---

## [v1.1.4] - 2026-06-16

### Added / 新增
- **角色专属代办事项（Todo）功能**：
  - 在每个角色列下方添加了独立的代办事项功能，采用流式标签展示，点击回车可快捷添加，点击标签上的 `×` 可手动删除。
  - 该代办属于角色独立数据，不受每日和每周重置清理影响，支持随着角色改名自动迁移，并可包含在每周历史存档中以供追溯。

---

## [v1.1.3] - 2026-06-16

### Added / 新增
- **数据自动刷新功能**：前端默认每 **10 分钟** 静默更新一次最新数据（如日常/周常数据的跨周期重置检测），提升挂机及长时间停留页面的使用体验。
- **智能输入避让机制**：当自动更新触发时，若检测到用户正处于数值或备忘录防抖编辑阶段（`saveDebounceRef.current` 计时器激活），刷新程序将主动避让并跳过本次拉取更新，保护用户尚未落盘的编辑成果不被覆盖。

### Refactored / 重构
- **Hooks 封装与优化**：将状态获取 `fetchStatus` 与授权请求 `authFetch` 分别使用 `useCallback` 重新封装。
- **级联渲染修复**：修复了在 `useEffect` 挂载期间同步触发 `setState` 导致的级联渲染与依赖项缺失等 ESLint 警告，代码实现达到 React 最佳规范。

---

## [v1.1.2] - 2026-06-16

### Added / 新增
- **PWA (渐进式网页应用) 支持**：
  - 新增 `manifest.json` 声明 PWA 应用配置，支持多规格图标。
  - 新增服务工作线程 `sw.js` (Service Worker) 以提供静态文件缓存并彻底放行动态 `/api/` 路由。
  - 允许在支持的浏览器（如 Chrome、Safari、Edge 等）中直接将考勤簿“安装成应用”至桌面或手机。
- **README 效果图展示**：在中文与英文版 README 顶部添加了“主界面”与“配置面板”相对路径并排展示截图表格。

- **配置面板悬浮布局**：重构 `.drawer-content` 的 CSS Grid/Flex 逻辑，实现了配置管理抽屉“标题固定在最上方不跟随滚动、中间配置表单独立滚动、底部保存按钮固定悬浮在下方”的精美排版，历史记录抽屉也同步应用了此项布局升级。

## [v1.1.0] - 2026-06-15

### Added / 新增
- **日常/周常重置行滚动固定**：在移动端或窄屏查阅时，重置行首列文本支持水平滚动粘性固定（Sticky Column），极大提升了高密度多角色对比的阅读感知。

## [v1.0.1] - 2026-06-15

### Added / 新增
- **角色改名与拖拽排序**：在线配置面板中，现在可以直接双击重命名已有角色，并可通过前置拖拽柄 `⋮⋮` 实现多角色的可视化拖拽位置排序，顺序即时持久化并同步重构本地 `data.json`。
- **行 Hover 高亮**：表格体行在鼠标滑过时呈现微妙的背景色变亮，有效降低长行表格对齐错位的查阅疲劳感。

---

## [v1.0.0] - 2026-06-15

### Added / 新增
- **多用户验证与物理隔离**：
  - 新增多用户支持。通过在系统运行前配置 `USERS_AUTH` 环境变量，支持不同用户以独立昵称和独立密码登录。
  - 新增数据物理隔离层。每个用户有其专属的 `config_${nickname}.json`、`data_${nickname}.json` 以及 `history_${nickname}.json`，实现完全的物理隔离。
  - 增加注销登出机制，顶栏一键登出并返回验证面。
- **兼容单人与免密模式**：
  - 兼容单人密码模式（由 `ADMIN_PASSWORD` 控制，昵称显示为“管理员”）。
  - 支持免密直接进入机制（昵称显示为“访客”）。
  - 浏览器通过 `LocalStorage` 记住登录凭证，免除重复输入。

## [v0.1.0] - 2026-06-15

### Added / 新增
- **阿尔特里亚大陆考勤簿 (altea-ledger) 首个稳定版发布**：
  - 支持多角色金币、时装套数及资产信息记录。
  - 提供黄金鹅 Buff 剩余时间设置和倒计时提示。
  - 每日任务（签到、喝药、日常等）、挂机进度（4阶段）打卡及智能日常重置（每日 09:00）。
  - 每周通关状态微调与手动加减，支持智能周常重置（周六 09:00）。
  - 周常重置前自动创建归档快照，支持侧边抽屉查阅最近 50 次周常历史表现。
